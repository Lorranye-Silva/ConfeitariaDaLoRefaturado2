Projeto Confeitaria Da Lo

Este repositório contém o código-fonte e documentação do Projeto Confeitaria Da Lo.

Descrição:

O sistema gerencia clientes, produtos, funcionários e vendas, com integração a banco de dados MySQL. Inclui validações no front-end e back-end para garantir a integridade dos dados.

Funcionalidades:

Cadastro de clientes, produtos e funcionários

Registro de vendas com controle de estoque

Validação de dados no front-end e back-end

Integração com banco de dados MySQL

Controle de versionamento via Git e GitHub

Tecnologias Utilizadas:

Java 17

Spring Boot

MySQL

Maven

Git
